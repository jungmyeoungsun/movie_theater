1. 정보구조 (Information Architecture)
영화 / 예매 / 극장 / 이벤트 / 멤버십 / 고객센터..로 메인 메뉴 구성
기능 중심 구조로 구성되어 있어 예매 접근성은 뛰어나지만 콘텐츠(영화) 중심이라기보다는 사용자가 이용하는 행동을 기반으로 단순화된 구조
메인 페이지 상단에 영상 재생 버튼을 배치해서 클릭시 바로 영상 재생(홍보효과↑)
*구조는 단순하고 효율적이지만, 콘텐츠 탐색 다양성은 부족*

2. 네비게이션 (Navigation)
상단 고정형 메뉴. 메인 배너 하단에는 인기 영화, 빠른 예매 버튼, 이벤트등 핵심 콘텐츠 바로가기가 배치되어 있음
안정적이지만 트랜디한 것과는 거리가 멀다.

모바일 앱은 하단 탭 메뉴 형식이 -> 홈 / 예매 / MY / 더보기 구조. 영화 상세 페이지에서 예매로 연결되는 흐름이 직관적임.
*영화 소개보다는 사용자가 예매하는 흐름을 단순화해서 제작한 느낌->실용성은 높지만 이외에 예매 목적 외에 사용하기에는 다소 아쉬움*

3. UI (User Interface)
색상/브랜드 컬러는 레드 & 블랙 중심에 배경이 흰색으로 되어있음-> 롯데 브랜드를 반영함.
배치 방식은 카드형 콘텐츠 + 배너형 콘텐츠가 혼합되어있으며, 영화 정보 페이지는 평점, 상영관 정보 등을 간결하게 제공해줌
예매는 상영시간->인원,좌석 선택->결제.. 등 단계별 프로세스가 명확하지만 시각적으로 흥미는 떨어질 수 있음(예매 흐름은 직관적이나 비주얼적 요소는 정적이라 흥미 요소가 적은 느낌)

4. 마케팅
이벤트 - 상시 예매 이벤트, 제휴 카드 할인, 포토카드 증정 등..
SNS 마케팅은 다소 약한 편임.

5. 디자인 스타일
실용적이고 브랜드 정체성을 유지하는데에 집중한 느낌(롯데:브랜드 컬러-레드,블랙)
영상, 인터렉티브 요소는 비교적 적은편에 속하는것같음. 포스터 위주로 정보를 제공해서 콘텐츠를 표현하여 콘텐츠 큐레이션이 미흡하다..(영화 이외 콘텐츠 경험을 제공하기에는 부족)





통영국제음악재단
1. 정보구조 (Information Architecture)
메인 메뉴 구성:
TIMF 소개 / 공연 / 아카이브 / 교육 / 커뮤니티 / 후원 / 참여하기 / 사이트맵

구조적 특징:

문화예술 콘텐츠 중심: 클래식 및 현대 음악 중심의 공연 및 교육 콘텐츠 제공

정보 중심: 공연 정보, 아카이브, 교육 프로그램 등 다양한 문화 콘텐츠 정보 제공

후원 및 참여 유도: 후원 및 참여하기 메뉴를 통해 방문자의 참여를 유도

관찰 포인트:

문화예술 기관으로서의 정체성을 강조하는 정보 구조

사용자 중심의 정보 제공보다는 기관의 활동과 콘텐츠 중심의 구성

2. 네비게이션 (Navigation)
웹 구조:

상단 고정형 메뉴

메인 배너 하단에 주요 공연 및 프로그램 정보 배치

각 메뉴 하위에 상세 페이지로의 직관적인 이동 가능

모바일 앱 구조:

모바일 최적화된 메뉴 구성

상단 메뉴와 하단 탭 메뉴를 통해 주요 콘텐츠로의 접근 용이

네비게이션 특징 요약:

전통적인 문화예술 기관의 웹사이트에 적합한 안정적인 구조

콘텐츠 중심의 탐색 흐름을 제공하며, 사용자 경험을 고려한 설계

3. UI (User Interface)
컬러/브랜딩:

세련된 톤의 컬러 팔레트 사용 (주로 블랙, 화이트, 그레이 계열)

TIMF 로고 및 아이콘을 활용하여 브랜드 정체성 유지

배치 구성:

그리드 기반의 콘텐츠 배치로 정보의 명확한 전달

이미지 중심의 콘텐츠 구성으로 시각적 흥미 유발

UX 요소:

단계별 정보 제공 (예: 공연 정보 → 예매 → 후기)

모바일 반응형 디자인 적용으로 다양한 디바이스에서의 접근성 확보

관찰 포인트:

문화예술 기관의 웹사이트로서의 품격 있는 디자인

시각적 요소와 정보 전달의 균형을 잘 맞춘 UI 구성

4. 마케팅
이벤트 운영:

정기적인 공연 및 프로그램을 통한 지속적인 콘텐츠 제공

후원 프로그램을 통해 기관의 활동에 대한 참여 유도

디지털 마케팅 전략:

SNS 채널을 통한 공연 정보 및 소식 공유

이메일 뉴스레터를 통한 정기적인 소식 전달

관찰 포인트:

문화예술 기관으로서의 신뢰성 있는 마케팅 전략 운영

디지털 채널을 통한 소통 강화 필요

5. 디자인 스타일
전반적 스타일:

미니멀하고 세련된 디자인으로 전문성과 품격 강조

문화예술 기관의 정체성을 반영한 디자인 요소 사용

비주얼 요소:

고해상도 이미지와 영상을 활용하여 공연의 분위기 전달

타이포그래피를 통한 정보의 계층 구조 명확화

감성/문화적 터치:

문화예술 콘텐츠의 감성적 요소를 디자인에 반영

사용자 경험을 고려한 디자인으로 접근성 및 편의성 제공
